Format reference

Question types QuizCraft understands

These markers match the included Word template.

Single choice

a. !Correct answerb. Other answer

Multiple answers

[*] Correct answer[ ] Other answer

Short answer

* Accepted answer 1* Accepted answer 2

Every * line is accepted as correct. A student only needs to enter one of them, and capitalization is ignored.

Numerical

= 3.14 +- 0.01= [3.13, 3.15]

Essay

____

Four or more underscores.

File upload

^^^^

Four or more carets.

Adding images

Use Word’s Insert → Pictures command and place the image directly in the question. QuizCraft preserves its displayed size and alternative text, then packages the image inside the QTI ZIP.

PNG, JPEG, GIF, WebP, SVG, TIFF, and BMP files are supported. Embed the file in the document—images linked from a website or another file are not included.

Also supported: GROUP / END_GROUP item banks, per-question points and titles, answer feedback, equations, images, and tables.
